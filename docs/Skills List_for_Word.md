Skills List（技能列表）——中文版
================================

> 说明：本文件整理了当前环境中可用的所有 Skill、功能简介及典型使用场景，格式采用“章节编号 + 小节编号”，便于在 Word 中直接使用多级列表与样式。建议在本机用 Word 打开本文件（或复制内容），然后另存为 `Skills List.docx`。

1. 临床决策与报告类 Skills
-------------------------

1.1 clinical-decision-support  
- 功能：生成专业的临床决策支持（CDS）文档，用于药物研发和临床研究。  
- 主要能力：  
  - 患者队列分析（按生物标志物分层、结局分析）。  
  - 生存分析与可视化（风险比、Kaplan-Meier 曲线、瀑布图等）。  
  - 基于循证指南的治疗推荐与决策路径。  
  - 符合监管要求的决策算法说明，支持 LaTeX/PDF 级别排版。  
- 典型使用场景：  
  - 在药企内部评估新药候选的临床获益与风险，生成决策汇报。  
  - 针对特定生物标志物（如 PD-L1、EGFR）分层分析治疗效果。  
  - 在培训中演示“从数据到临床决策支持报告”的完整流程。

1.2 clinical-reports  
- 功能：撰写和规范化各类临床报告与病历文书，强调隐私与合规。  
- 主要能力：  
  - 病例报告（符合 CARE 指南），用于期刊投稿。  
  - 诊断报告：放射科、病理、实验室检验报告等。  
  - 临床试验报告：严重不良事件（SAE）、临床研究报告（CSR，符合 ICH‑E3、ICH‑GCP、FDA 等要求）。  
  - 临床文书：SOAP 病程记录、入院记录（H&P）、出院小结、会诊记录等。  
  - 提供隐私脱敏（HIPAA）、监管合规和质控检查清单。  
- 典型使用场景：  
  - 指导住院医师按规范撰写病例报告，用于教学或投稿。  
  - 为临床试验团队生成 SAE 报告模板和完整 CSR 结构。  
  - 培训新员工如何书写合规的出院小结和病程记录。

2. 临床与基因组数据库类 Skills
-----------------------------

2.1 clinicaltrials-database  
- 功能：通过 ClinicalTrials.gov API v2 检索和获取临床试验信息。  
- 主要能力：  
  - 按疾病、药物、地区、试验阶段、招募状态等检索试验。  
  - 通过 NCT 编号获取单个试验的详细信息。  
  - 导出试验数据，用于竞品分析、设计新研究、患者匹配等。  
- 典型使用场景：  
  - 在课程中演示如何查找某一疾病领域的正在进行的试验。  
  - 为新药立项做竞品临床试验格局分析。  
  - 帮助临床医生为患者筛选潜在适配的临床试验。

2.2 clinpgx-database  
- 功能：访问 ClinPGx 药物基因组数据库，支持精准用药决策。  
- 主要能力：  
  - 查询基因‑药物相互作用及药物基因组学证据。  
  - 查阅 CPIC 指南，获取基于基因型的给药建议与剂量调整。  
  - 了解等位基因功能（正常/减弱/丧失等）及其临床意义。  
- 典型使用场景：  
  - 在教学中示范“基因型指导用药”（如华法林、氯吡格雷剂量调整）。  
  - 为药师和医生提供个体化给药推荐。  
  - 设计药物基因组学相关的科研课题或培训案例。

2.3 clinvar-database  
- 功能：利用 NCBI ClinVar 数据库进行遗传变异的临床解释。  
- 主要能力：  
  - 按基因或具体变异位点查询临床意义（致病、良性、意义未明等）。  
  - 使用 API 或 FTP 对 VCF 等变异文件进行批量注释。  
  - 支持临床诊断、遗传咨询与科研分析中的变异解读。  
- 典型使用场景：  
  - 在遗传病案例教学中，演示如何查找某一突变的临床意义。  
  - 帮助遗传咨询师解释检测报告中的关键变异。  
  - 为科研团队批量注释测序数据中的候选位点。

2.4 cosmic-database  
- 功能：访问 COSMIC 癌症突变数据库，用于肿瘤学研究。  
- 主要能力：  
  - 检索癌症相关体细胞突变与 Cancer Gene Census。  
  - 分析突变谱、突变特征、基因融合等。  
  - 支持精准肿瘤治疗、靶点发现与药物研发（需认证访问）。  
- 典型使用场景：  
  - 在肿瘤学课程中展示某一肿瘤类型的高频突变谱。  
  - 帮助研究者评估某一基因是否为“癌症驱动基因”。  
  - 为靶向药物或免疫治疗设计提供分子依据。

3. 数据处理与大数据分析类 Skills
-------------------------------

3.1 dask  
- 功能：为 pandas/NumPy 工作流提供分布式与超大数据支持。  
- 主要能力：  
  - 将现有 pandas 代码扩展到多核或集群环境。  
  - 处理大于单机内存的大型 CSV/Parquet 等数据集。  
  - 支持分布式特征工程、机器学习流水线和批量文件处理。  
- 典型使用场景：  
  - 培训数据团队如何从单机 pandas 迁移到分布式计算。  
  - 分析包含数千万行记录的大规模临床或医保数据。  
  - 在教学中展示任务图（task graph）与惰性执行概念。

3.2 exploratory-data-analysis  
- 功能：对科学数据文件执行自动化探索性数据分析（EDA）。  
- 主要能力：  
  - 自动识别 200+ 科学数据格式的结构与内容。  
  - 生成详细的 Markdown 报告：包括字段结构、分布、缺失值、质量评估等。  
  - 提出针对性的数据清洗和后续分析建议，适合实验与临床数据初步摸底。  
- 典型使用场景：  
  - 在实训课中快速“扫一眼”学生提交的各类实验数据文件。  
  - 对新拿到的临床试验原始数据做质量和结构评估。  
  - 作为 EDA 教学案例，展示自动化数据体检流程。

3.3 polars  
- 功能：使用 Polars DataFrame 库进行高速内存内数据处理。  
- 主要能力：  
  - 提供比 pandas 更快的查询和聚合性能。  
  - 支持惰性执行、并行计算和 Arrow 后端。  
  - 适合 1–100 GB 范围内、可放入内存的数据分析与 ETL 管道。  
- 典型使用场景：  
  - 在课程中对比 pandas 与 Polars 的性能差异。  
  - 构建需要高性能聚合和联结操作的分析流水线。  
  - 教授学生如何使用惰性查询优化数据处理流程。

3.4 vaex  
- 功能：针对“超出内存”的超大表格数据进行高效分析。  
- 主要能力：  
  - 处理可达数十亿行的 CSV/HDF5/Arrow/Parquet 数据。  
  - 支持惰性计算、快速统计与可视化。  
  - 构建不需要全部载入内存的大数据机器学习流程。  
- 典型使用场景：  
  - 在大数据课程中演示如何对十亿级行数据进行交互分析。  
  - 对全国范围多中心队列数据做探索和统计。  
  - 作为“超出内存数据分析”教学的实践工具。

四、文献与外部信息检索类 Skills
-----------------------------

11. openalex-database  
    - 功能：利用 OpenAlex 数据库检索和分析学术文献。  
    - 主要能力：  
      - 检索特定主题、作者、机构的论文。  
      - 分析研究趋势、引用网络和学术影响力。  
      - 查找开放获取（Open Access）文献和进行计量学分析。

12. pubmed-database  
    - 功能：通过 E-utilities API 访问 PubMed 医学文献数据库。  
    - 主要能力：  
      - 使用 MeSH 词表与布尔检索进行系统文献搜索。  
      - 批量下载检索结果，管理 PMID、摘要和引用信息。  
      - 为系统评价、临床指南制定和科研立题提供文献基础。

13. perplexity-search  
    - 功能：调用 Perplexity 模型进行带来源引用的实时网络搜索。  
    - 主要能力：  
      - 获取最新的科研进展与网络信息，突破模型知识截止时间。  
      - 对复杂问题进行多步推理和聚合性搜索，并返回引用链接。  
      - 支持科学研究、技术调研和时效性要求较高的任务。

14. fda-database  
    - 功能：通过 openFDA API 访问 FDA 监管相关公开数据。  
    - 主要能力：  
      - 查询药品、医疗器械的不良事件与安全信号。  
      - 检索召回信息、标签说明书、510(k)/PMA 申报信息。  
      - 支持药物/器械安全性评估、监管合规和市场分析。

15. markitdown  
    - 功能：将多种文档和媒体格式转换为 Markdown 文本。  
    - 主要能力：  
      - 支持 PDF、DOCX、PPTX、XLSX、CSV、JSON、XML、HTML、ZIP、EPUB 等。  
      - 支持图片 OCR、音频转写以及 YouTube 等链接内容提取。  
      - 便于在项目中统一为可编辑的 Markdown 格式，方便进一步处理和版本控制。

五、机器学习与模型解释类 Skills
-----------------------------

16. scikit-learn  
    - 功能：使用 scikit-learn 完成传统机器学习建模与评估。  
    - 主要能力：  
      - 分类、回归、聚类、降维等常见算法。  
      - 数据预处理、特征工程与模型管道（Pipeline）。  
      - 模型评估、交叉验证和超参数搜索（Grid/Random Search 等）。

17. scikit-survival  
    - 功能：利用 scikit-survival 进行生存分析和时间‑至‑事件建模。  
    - 主要能力：  
      - 构建 Cox 比例风险模型、随机生存森林、梯度提升生存模型等。  
      - 处理删失数据（censoring），进行竞争风险分析。  
      - 使用一致性指数（C‑index）、Brier score 等指标评估模型性能。

18. shap  
    - 功能：使用 SHAP 框架解释机器学习模型的预测结果。  
    - 主要能力：  
      - 适用于树模型（XGBoost、LightGBM、随机森林）、深度学习、线性模型和黑盒模型。  
      - 生成多种可视化：waterfall 图、beeswarm 图、force plot、热图等。  
      - 分析特征重要性、检测模型偏倚和公平性问题，帮助调试和优化模型。

19. pyhealth  
    - 功能：面向医疗健康数据的综合 AI 工具箱。  
    - 主要能力：  
      - 处理电子病历（EHR）、医疗编码（ICD、NDC、ATC 等）、生理信号（EEG、ECG）等。  
      - 针对再入院预测、死亡率预测、用药推荐等任务构建模型。  
      - 集成多种深度学习结构，如 RETAIN、Transformer、图神经网络（GNN）等。

20. statistical-analysis  
    - 功能：为统计分析提供检验选择和结果报告指导。  
    - 主要能力：  
      - 根据数据类型与研究设计推荐合适的统计检验方法。  
      - 检查假设前提（正态性、方差齐性等）并进行功效分析。  
      - 生成符合 APA 格式的统计结果描述，便于论文撰写。

21. treatment-plans  
    - 功能：生成聚焦且简明的医学治疗方案文档（约 3–4 页）。  
    - 主要能力：  
      - 涵盖各专科的一般治疗、康复治疗、精神健康、慢病管理、围手术期和疼痛管理等。  
      - 使用 SMART 目标框架与循证干预措施。  
      - 输出专业排版的 LaTeX/PDF 文档，突出临床可操作性与合规性。

六、Cursor 配置与扩展类 Skills
---------------------------

22. create-rule（skills-cursor）  
    - 功能：为 Cursor 项目创建和维护规则与约定。  
    - 主要能力：  
      - 生成和更新 RULE.md 等项目级规则文件。  
      - 记录编码规范、文件结构约定和智能体协作准则。  
      - 帮助在多人或多智能体环境中保持行为一致性。

23. create-skill（skills-cursor）  
    - 功能：指导用户编写新的 Skill（SKILL.md）。  
    - 主要能力：  
      - 提供 Skill 文件的标准结构模板。  
      - 帮助梳理任务、工具权限和使用说明，将个人工作流封装为可复用 Skill。  
      - 支持在 Cursor 中扩展定制化自动化能力。

24. update-cursor-settings（skills-cursor）  
    - 功能：修改 Cursor/VSCode 的用户设置（settings.json）。  
    - 主要能力：  
      - 配置编辑器主题、字体、缩进、自动保存与自动格式化策略。  
      - 定制快捷键、编辑行为和代码风格偏好。  
      - 统一和优化本地开发环境体验。
